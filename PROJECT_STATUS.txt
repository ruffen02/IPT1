QUICK START
═══════════════════════════════════════════════════════════════════════════════

1. Create Database:
   mysql -u root -e "CREATE DATABASE IF NOT EXISTS ipt;"

2. Install Dependencies:
   cd c:\Users\Acer\Desktop\niekoipt\ipt
   composer install

3. Run Migrations:
   php artisan migrate

4. Start Development Server:
   php artisan serve

5. Open in Browser:
   http://localhost:8000/products

   ✨ Professional table with all products displayed!

DATABASE SCHEMA
═══════════════════════════════════════════════════════════════════════════════

CATEGORIES TABLE:
┌──────────┬──────────┬────────────────┐
│ Column   │ Type     │ Constraints    │
├──────────┼──────────┼────────────────┤
│ id       │ BIGINT   │ PK, AI         │
│ name     │ VARCHAR  │ Required       │
│ desc.    │ LONGTEXT │ Nullable       │
│ created  │ TIMESTAMP│ Auto           │
│ updated  │ TIMESTAMP│ Auto           │
└──────────┴──────────┴────────────────┘

PRODUCTS TABLE:
┌──────────┬──────────┬────────────────┐
│ Column   │ Type     │ Constraints    │
├──────────┼──────────┼────────────────┤
│ id       │ BIGINT   │ PK, AI         │
│ name     │ VARCHAR  │ Required       │
│ desc.    │ LONGTEXT │ Nullable       │
│ price    │ DECIMAL  │ Required, >0   │
│ qty      │ INT      │ Required, >0   │
│ cat_id   │ BIGINT   │ FK, Required   │
│ created  │ TIMESTAMP│ Auto           │
